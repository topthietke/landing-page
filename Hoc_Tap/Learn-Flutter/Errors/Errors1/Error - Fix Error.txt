Cách xử lý: 
    Lỗi Visual Studio Toolchain
    Lỗi này xảy ra vì Flutter trên Windows cần Visual Studio (không phải VS Code) với C++ build tools để chạy ứng dụng desktop.
    Cách fix:
    1. Cài Visual Studio (miễn phí)

    Tải Visual Studio Community tại visualstudio.microsoft.com
    Trong quá trình cài, chọn workload: "Desktop development with C++"

    2. Sau khi cài xong, kiểm tra:
    bashflutter doctor
    3. Nếu vẫn báo lỗi, chạy thêm:
    bashflutter doctor --android-licenses

    Hoặc nếu bạn chỉ muốn chạy trên Android/Web (không cần VS):
    bash# Chạy trên Chrome
    flutter run -d chrome

    # Chạy trên Android emulator/device
    flutter run -d android

    Lưu ý thêm:
    Bạn cũng có 4 packages cũ hơn, có thể cập nhật bằng:
    bashflutter pub upgrade
    Bạn đang muốn chạy app trên Windows desktop, Android, hay Web?
