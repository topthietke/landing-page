Mixins là một khái niệm ngôn ngữ cho phép lập trình viên đưa một số mã vào một lớp. Lập trình mixin là một phong cách phát triển phần mềm, trong đó các đơn vị chức năng được tạo ra trong một lớp và sau đó trộn lẫn với các lớp khác. Lớp mixin hoạt động như lớp cha, chứa chức năng mong muốn.

mixin Example2 on Person{
  void ex3();
  void ex4();
}

Trong mixin sử dụng on đến class cha thì các class con nào extens class cha thì mới sử dụng được with đến mixin;
